Лабораторна робота №7

Створення сайту GitHub Pages

Порядок виконання лабораторної роботи №7:

  1. Уважно ознайомтеся з загальними теоретичними положеннями щодо служби GitHub Pages
  2. Створити репозиторію у GitHub для сайту.
  3. Створити файл запису для сайту. Як файл запису для вашого сайту GitHub Pages буде використовуватись index.html та додаткові файли скриптів та стилей. У файлі відобразити ФІО та № групи студента.
  4. Запустити створену веб-сторінку на апач сервері, що використовувався у лр№5.
